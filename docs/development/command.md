# 開発環境コマンド

## RSpec

Rspec を実行し、テストを実行します。

```
bin/rspec
```

## Rubocop

Rubocop を実行し、コードの静的解析をします。

```
bin/rubocop
```

## JavaScript/TypeScript Lint

npm run lint を実行し、JavaScript/TypeScript ファイルの静的解析をします。

```
npm run lint
```

## JavaScript/TypeScript Test

npm run test を実行し、テストを実行します。

```
npm run test
```

## TypeScript Type Check

npm run typecheck を実行し、型チェックを実行します。

```
npm run typecheck
```
